#  Projeto Academia

Sistema web simples desenvolvido para gerenciamento de uma academia.

##  Sobre o projeto
Este projeto foi desenvolvido com o objetivo de praticar desenvolvimento web utilizando PHP e CSS.  
O sistema contém telas básicas de autenticação e organização de usuários.

##  Funcionalidades
- Tela de login
- Tela de cadastro
- Página de aluno
- Página de professor
- Conexão com banco de dados

##  Tecnologias utilizadas
- PHP
- CSS
- HTML
- MySQL

##  Estrutura do projeto
- `login.php` → Tela de login
- `cadastro.php` → Tela de cadastro
- `pgaluno.php` → Área do aluno
- `pgprof.php` → Área do professor
- `conexao.php` → Conexão com o banco de dados
- `*.css` → Estilização das páginas

## 💻 Como executar o projeto
1. Instale o XAMPP ou outro servidor local
2. Coloque o projeto na pasta `htdocs`
3. Inicie o Apache e MySQL
4. Acesse no navegador:
5. http://localhost/Projeto_academia
